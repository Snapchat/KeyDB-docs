KeyDB offers a variety of packages for easy installation accross different distributions and architectures. 

## KeyDB Managed Packages

The packages and distribution methods outlined below are actively managed by KeyDB.

| Type | Distribution Method Description  |Link |
|----------------------------------|------------------|---------|
| <img src="/img/download/docker.svg" style={{width: "100px"}}/> | Use our Docker Image to run your own containers. Follow this link to visit our docker page outlining pull and usage options. This image is manifested and supports x86 & ARM archtectures, a simple `docker pull eqalpha/keydb` will pull correct version. | <a href="https://hub.docker.com/r/eqalpha/keydb"><button class="button1">Docker</button></a> |
| <img src="/img/download/ubuntu.svg" style={{width: "100px"}}/> | Using Ubuntu? KeyDB has DEB packages for **Focal/Bionic/Xenial** packages for both x86_64/ARM_64 architectures. This includes binary and service installations. Just add the KeyDB Freight PPA to your /etc/apt/sources.list.d to be able to 'apt install keydb`. Click this link for full instructions, or go to the source link for individual .deb downloads. | <a href="/docs/ppa-deb"><button class="button1">PPA</button></a><br/><a href="https://download.keydb.dev/pkg/open_source/deb/" style={{padding: "10%"}}>source dir</a> |
| <img src="/img/download/debian.svg" style={{width: "100px"}}/> | KeyDB Debian packages are now available through the KeyDB Freight repo. **Bullseye/Buster/Stretch** packages for x86_64 & ARM64 are available. Similar to Ubuntu, add the repo to your /etc/apt/sources.list/d to be able to `apt install keydb`. You can also go to the sources link to download .deb packages individually. | <a href="/docs/ppa-deb"><button class="button1">DEB</button></a><br/><a href="https://download.keydb.dev/pkg/open_source/deb/" style={{padding: "10%"}}>source dir</a> |
| <img src="/img/download/centos.svg" style={{width: "100px"}}/> | **Centos7/8** RPM packages are available for individual download (x86_64 & ARM64). Click the button for instructions or the source link to go directly to the RPM package directory. The RPM packages will install the binaries as well as the service files. | <a href="/docs/rpm"><button class="button1">RPM</button></a><br/><a href="https://download.keydb.dev/pkg/open_source/rpm/" style={{padding: "10%"}}>source dir</a> |
| <img src="/img/download/wrench.png" style={{width: "100px"}}/> | For instructions on building KeyDB yourself select this option. You will find instructions for building on Centos 7/8, Ubuntu/Debian, Archlinux, Alpine and Fedora. Additional notes on build flags, and packaging dependencies are also here. | <a href="/docs/build"><button class="button1">Build</button></a> |
| <img src="/img/download/github.png" style={{width: "100px"}}/> | Go directly to the github page to see the sourcecode, create and issue or PR, or clone and download from there. | <a href="https://github.com/EQ-Alpha/KeyDB"><button class="button1">Github</button></a><br/> |

## Community Contributed Offerings

The KeyDB Community has created and is managing some great distribution channels! Please see options below. 

| Type | Distribution Method Description |Link |
|----------------------------------|------------------|---------|
| <img src="/img/download/homebrew.svg" style={{width: "100px"}}/> | **Homebrew formulae** for KeyDB is available. Click this option to go to the Homebrew page, or simply run `brew install keydb`. Support avaialble for Intel (Monterey/Big Sur/Catalina/64-Bit Linux) and Apple Silicon(Monterey/Big Sur). | <a href="https://formulae.brew.sh/formula/keydb"><button class="button1">HomeBrew</button></a><br/><a href="https://github.com/Homebrew/homebrew-core/blob/HEAD/Formula/keydb.rb" style={{padding: "10%"}}>source dir</a> |

Select this option to view on ArtifactHUB of source for the github link. | <a href="https://artifacthub.io/packages/helm/enapter/keydb"><button class="button1">Helm</button></a><br/><a href="https://github.com/Enapter/charts" style={{padding: "10%"}}>source dir</a> |
